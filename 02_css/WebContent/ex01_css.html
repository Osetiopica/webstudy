선택자: 표시
전체 선택자: *
태그 선택자: 해당 태그
아이디 선택자: #
클래스 선택자: .
자식 선택자: 부모태그>자식태그
자손 선택자: 부모태그 자식태그
요소 속성 선택자: 선택자[속성=값] => 해당 속성과 일치하는 요소 선택
			 선택자[속성^=값] => 해당 속성으로 시작하는 요소 선택
			 선택자[속성*=값] => 해당 속성을 포함하는 요소 선택
			 선택자[속성$=값] => 해당 속성으로 끝나는 요소 선택
	구조 선택자: 선택자:first => 첫 번째 요소 선택
			  선택자:last => 마지막 요소 선택
			  선택자:even => 짝수 요소 선택
			  선택자:odd => 홀수 요소 선택
